CLI-CoinMarketCap
=================
This example consumes a [CoinMarketCap](https://coinmarketcap.com/) api and returns a coin's data.

### Build & Run
```
$ go build

$ ./coinmarketcap ethereum
 
[CoinMarketCap] [ethereum]
---------------------------
Price(BTC):   0.0440075
Price(USD):   328.033
Volume:       601065000.0
Change 24H:   -0.88
```
